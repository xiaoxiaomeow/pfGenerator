盟军斗篷 (Allied Cloak)
学派            防护  
环位            吟游诗人/歌者 3、血脉狂怒者 3、魔战士 3、秘学士 2、术士/法师/奥能师 3、召唤师 3
施法时间            标准动作
成分            语言、姿势、器材 (斗篷或外衣)
范围            个人
目标            自己
持续时间            1轮/等级
豁免            无
法术抗力            无
你使一件斗篷(cloak)、披肩、斗篷(poncho)或其他你穿戴着的外衣活化，援助与守护你。这斗篷对你的AC提供+2的盾牌加值。此外，每轮一次，在你的回合内，你可以以一个自由动作，指挥你的斗篷使用援助他人行动来协助你的技能检定、攻击骰或AC。斗篷的攻击加值或总技能加值，等同你的施法者等级+施法关键属性调整值(吟游诗人是魅力、法师是智力，以此类推)。如果在法术持续时间中，另一个生物穿戴着斗篷，该斗篷会提供它的保护与援助给穿戴者，但斗篷在使用援助他人行动上，使用同样的加值，如同你还是穿戴者。这斗篷不能采取任何其他种类的行动。
译注：cloak、poncho都是斗篷，但cloak就是一般我们常看到的，披在身后的那种; poncho则是南美人穿戴的，会罩住上半身，手下从下方伸出的那种。
华裙翩翩 (Billowing Skirt)
学派            变化  
环位            吟游诗人/歌者 2、审判者 2、秘学士 2、萨满 2、术士/法师/奥能师 2、女巫 2
施法时间            标准动作
成分            语言、姿势、器材 (一件裙子、短裙或长袍)
范围            个人
目标            自己
持续时间            1分钟/等级
豁免            无
法术抗力            无
一件妳穿着的裙子、短裙或长袍令你能够控制你在空中的坠落。一名生物身着被附上魔法的衣物，坠落的速度变慢，如同受到羽落术(feather fall)的影响，并且能以一个移动动作，尝试一次DC15飞行技能检定，来盘旋一轮。
攀爬抓勾 (Grappling Scarf)
学派            变化  
环位            吟游诗人/歌者 1、审判者 1、秘学士 1、术士/法师/奥能师 1
施法时间            标准动作
成分            语言、姿势、器材 (围巾)
范围            个人
目标            自己
持续时间            1分钟/等级
豁免            无
法术抗力            无
一条你穿着的围巾磨损，最终变成勾状，并可以延长到不可思议的长度。尽管你的勾子重量跟围巾一样，在磨损结束后如爪勾(grappling hook)般运作，并可以用来扔掷，射程增量为20尺。在此法术持续时间内，你的勾子在首次扔掷可以伸长至100尺。不论材料为何，勾子足够强韧，并能轻易用来攀爬，就好像打好结的丝绳。在法术持续时间结束时，勾子分离并且返回先前的形状，如果你还没准备好取回它，它会在它的位置上缓缓飘落。
安逸斗篷 (Restful Cloak)
学派            咒法 (治疗) 
环位            牧师/先知/战斗祭司 2、德鲁伊/猎人 2、秘学士 2、圣武士 2、游侠 2、萨满 2
施法时间            标准动作
成分            语言、姿势、器材 (斗篷)
范围            个人
目标            自己
持续时间            1小时/等级
豁免            无
法术抗力            无
你穿着的一件斗篷状变成一顶帆布帐篷，大到能够容纳四名中体型生物，牢牢的搭在临接你的一处平坦区域(如果没有那种区域，此法术失败)。当在这帐篷休息时，除了其他任何休息的效益外，一名生物每小时重获1点生命值。此外，一名疲乏(fatigued)的生物在帐篷内休息一小时后会失去该疲乏状态，但一次安逸斗篷只能恢复该生物一次疲乏状态。在此法术持续时间结束时，帐篷变回斗篷。如果帐篷在法术持续时间到期之前被移动了，则此法术立即结束。
塑形围巾 (Sculpted Cape)
学派            变化  
环位            吟游诗人/歌者 1、秘学士 1、游侠/猎人 1、萨满 1、术士/法师/奥能师 1、女巫 1
施法时间            标准动作
成分            语言、姿势、器材 (披肩，或宽松的外衣)
范围            个人
目标            自己
持续时间            1分钟/等级
豁免            无
法术抗力            无
你的围巾、斗篷或披肩硬化并且就算被移开也保持着它的形状。一件塑型围巾有着2点硬度以及等同你施法者等级的生命值，并且可以支撑每施法者等级/1磅重。一件未被穿着的塑形围巾可以提供掩蔽(cover)。
适足之靴 (Surefoot Boots)
学派            变化  
环位            德鲁伊/猎人 2、秘学士 2、游侠 2、萨满 2
施法时间            标准动作
成分            语言、姿势、器材 (鞋类)
范围            个人
目标            自己
持续时间            1轮/等级
豁免            无
法术抗力            无
你碰触了你的靴子、鞋子或其他鞋类并对他们注入就算在最险峻的地形，也能牢牢抓稳的能力。当处于此法术的效果时，该鞋类的穿戴着在对待任何有关平衡的特技技能检定时，就好像穿戴者骰出了一颗20。
